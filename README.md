# Compose Catalog & Animations

A collection of practical examples covering Jetpack Compose components, animations, and core concepts. All examples are accessible directly from the app menu.

---

## ✨ Key Features
Centralized Navigation: A smooth Navigation Drawer to browse through different UI patterns seamlessly.

Custom Animations: Practical examples of AnimatedVisibility, animateContentSize, and complex transitions.

Scalable Architecture: Organized with a clean navigation graph, making it easy to discover and reuse components.

---

## 📂 Project Structure
**`components/`:** A collection of many UI elements like Buttons, Cards, Text, Dialogs, and more, showing different ways to build interfaces in Compose.

**`ui/concepts/`:** Modules dedicated to applied theory:

* **State:** Practical examples using remember, rememberSaveable, and state management.

* **Advanced:** Side Effects (LaunchedEffect), performance optimization with derivedStateOf, and InteractionSource handling.

* **Layout:** A visual guide on positioning using Box, Column, Row, and ConstraintLayout.

* **Navigation:** Simple implementations of screen-to-screen transitions.

**`ui/exercises/`:** Real-world use cases and complex structures, including advanced Scaffolding, Snackbars, and mixed layouts.

**`ui/navigation/`:** The main navigation logic, featuring a Modal Drawer to navigate between all examples in the app.
